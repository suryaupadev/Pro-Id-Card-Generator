Pro ID Card Generator 💳

A sleek, Visa-style professional ID card generator built with vanilla HTML, CSS, and JavaScript. This project focuses on Glassmorphism UI, 3D interactions, and dynamic asset generation.

🚀 Live Demo
/Demo will be updated soon/

✨ Features
Minimalist Input Form: Clean user flow for data entry.
Visa-Style UI: Obsidian gradient background with glassmorphism effects.
3D Parallax Hover: Interactive card tilt effect using CSS transforms.
Download Functionality: Saves the generated card as an image directly to your device.
🛠️ Built With
HTML5 - Semantic Structure
CSS3 - Flexbox, Grid, Glassmorphism, 3D Transforms
JavaScript - DOM Manipulation, Canvas API (for image download)

🏃 How to Run Locally
Clone the repo: git clone https://github.com/yourusername/pro-id-generator.git
Open index.html in your browser.
